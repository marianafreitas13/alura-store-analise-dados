# 📊 Análise de Dados - Alura Store

## 📌 Propósito da Análise

Este projeto tem como objetivo realizar uma **análise exploratória de dados (EDA)** das vendas das quatro lojas da **Alura Store**.

A análise foi desenvolvida para auxiliar o **Sr. João**, proprietário das lojas, a tomar uma decisão estratégica sobre **qual loja vender** para investir em um novo negócio.

Através da análise de dados de vendas, avaliações de clientes e custos de frete, foi possível identificar padrões de desempenho entre as lojas e gerar recomendações baseadas em dados.

# 📊 Análises Realizadas

Durante o projeto foram realizadas diversas análises para comparar o desempenho das lojas.

## 1️⃣ Faturamento por Loja

Foi calculado o faturamento total de cada loja para identificar qual possui maior geração de receita.

**Resultado obtido:**

| Loja   | Faturamento     |
| ------ | --------------- |
| Loja 1 | R$ 1.534.509,12 |
| Loja 2 | R$ 1.488.459,06 |
| Loja 3 | R$ 1.464.025,03 |
| Loja 4 | R$ 1.384.497,58 |

📌 **Insight:**
A **Loja 1 apresenta o maior faturamento**, enquanto a **Loja 4 possui o menor desempenho financeiro**.

---

## 2️⃣ Categorias de Produtos Mais Vendidas

A análise das categorias permitiu identificar quais tipos de produtos possuem maior demanda entre os consumidores.

📌 **Insight:**
Algumas categorias apresentam maior volume de vendas, indicando oportunidades estratégicas para aumento de estoque e campanhas de marketing.

---

## 3️⃣ Avaliação Média dos Clientes

As avaliações foram analisadas para medir o nível de **satisfação dos clientes**.

📌 **Insight:**
Lojas com melhores avaliações tendem a apresentar maior confiança dos consumidores e maior potencial de fidelização.

---

## 4️⃣ Produtos Mais Vendidos

A identificação dos produtos mais vendidos permite compreender as preferências do público e direcionar melhor as estratégias de vendas.

📌 **Insight:**
Produtos com alta demanda podem ser utilizados em promoções e campanhas para aumentar ainda mais o faturamento.

---

## 5️⃣ Custo Médio de Frete

O frete também foi analisado, pois pode influenciar diretamente a decisão de compra dos clientes.

📌 **Insight:**
Custos de frete mais elevados podem reduzir a competitividade da loja.

---

# 📈 Exemplos de Gráficos

Durante a análise foram criadas visualizações utilizando **Matplotlib** para facilitar a comparação entre as lojas.

Exemplos de gráficos utilizados:

* Gráfico de **faturamento por loja**
* Gráfico de **categorias mais vendidas**
* Gráfico de **avaliação média das lojas**
* Gráfico de **produtos mais vendidos**

Essas visualizações ajudam a identificar padrões e diferenças de desempenho entre as unidades.

---

# 🧠 Recomendação Final

Com base nas análises realizadas, recomenda-se que o **Sr. João venda a Loja 4**.

Essa recomendação foi feita porque:

* A **Loja 4 apresentou o menor faturamento total** entre todas as unidades.
* Outras lojas demonstraram maior capacidade de geração de receita.
* Manter as lojas com melhor desempenho pode trazer maior retorno financeiro.

Dessa forma, vender a Loja 4 pode permitir que o Sr. João concentre seus investimentos nas unidades mais lucrativas.
